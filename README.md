# **Banible**
### 유아 움직임 감지 웨어러블 밴드
<br>

## 💡 **Project Info**

>📅 기간 : 2019.09 ~ 2019.12

> - 경영, 글로벌 서비스, 산업디자인 전공으로 이루어진 **팀 프로젝트**<br>
> - Design Thinking 기법을 적용한 프로젝트<br>
> - 주제 선정(공통), **아두이노를 이용한 기능 구현** 담당
<br>
